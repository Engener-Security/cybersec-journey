# Cybersec Journey — Márcio

Documentação pública da minha aprendizagem em Cybersecurity, Networking e Análise de Dados.

## Sobre mim

Tenho 20 anos, vivo em Portugal e estou a frequentar uma formação profissional em Cybersecurity e Análise de Dados no Prepara Portugal. Trabalho na área industrial enquanto estudo. Este repositório é onde documento tudo o que aprendo — labs, notas, scripts e projectos.

## Objectivo

Construir competências sólidas em network security e análise de dados de segurança, com o foco inicial em:

- Fundamentos de redes (TCP/IP, DNS, protocolos)
- Linux e linha de comandos
- Análise de tráfego e logs
- Ferramentas de segurança ofensiva e defensiva (Kali Linux, Wireshark, Nmap)

Meta a 12 meses: conseguir o primeiro projecto real ou posição na área.
